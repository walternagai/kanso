# Kanso CLI

> Build static sites with quiet speed.

Kanso CLI is a modern, simple, and fast static site generator for HTML, CSS, JavaScript, and Markdown. No heavy client-side frameworks — just clean, portable output.

## Status

🚧 In planning — PRD and PRPs completed, implementation pending.

## Quick Start

```bash
# Install (coming soon)
npm install -g kanso

# Create a new project
kanso init my-site

# Start dev server
cd my-site
kanso dev

# Build for production
kanso build

# Deploy
kanso deploy
```
